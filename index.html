🌐 Web Nedir?

Web: internet üzerinden erişilebilen sayfa ve uygulamaların genel adıdır. Web yapısı istemci (client) ve sunucu (server) mantığıyla çalışır.

👩‍🏫 Hoca Örneği:

Kullanıcı → Tarayıcıdan siteye girer → Sunucu isteği alır → Cevap gönderir

🎨 Frontend (Ön Yüz)

Kullanıcının gördüğü kısımdır, tarayıcıda çalışır. Teknolojiler: HTML, CSS, JavaScript

⚙️ Backend (Arka Yüz)

Sunucuda çalışan kısımdır. Teknolojiler: PHP, ASP.NET, Java, Python, C#, Node.js

🗄️ Veritabanı (Database)

MySQL, PostgreSQL, MongoDB gibi veritabanı sistemleri kullanılır.

🌍 Internet

TCP/IP tabanlı herkese açık ağ. Dünya çapında bilgisayar ağlarını birbirine bağlar.

📡 TCP/IP

Transmission Control Protocol / Internet Protocol. Amerikan Savunma Bakanlığı tarafından hazırlanmıştır.

🏢 Intranet

Bir firmanın kendi ağı içindeki iletişim sistemidir. Sadece kurum içi erişime açıktır.

🕸️ World Wide Web

Web browser ile görüntülenebilen Internet'in grafiksel yüzüdür.

📄 Web Page

Internet üzerinde görüntülenebilen her dosya. HTML ile yazılır.

🌐 Web Browser

WWW üzerindeki sayfaları yükleyip görüntüleyen program (Chrome, Firefox, Edge).

📁 FTP

File Transfer Protocol. Dosya transfer protokolü. Örnek: FileZilla programı.

🔗 DNS

Domain Name Server. Alan adı ile IP adresi eşleştirmesini yapar.

🖥️ Web Server

HTML sayfalarını tarayıcıya gönderen program (Apache, Nginx, IIS).

📍 URL

Internet adreslerinin genel adı. Örnek: http://www.artuklu.edu.tr/

🏷️ Alan Adı Uzantıları

.com → Commercial (Ticari) .gov → Government (Devlet) .org → Organization (Organizasyon) .edu → Education (Eğitim) .net → Network (Ağ) .info → Information (Bilgi)
1️⃣ Kullanıcı hyperlink'e tıklar
2️⃣ Browser DNS'e bağlanır, IP adresini alır
3️⃣ IP'ye çağrı gönderir, sayfa istenir
4️⃣ Web Server isteğe cevap verir
5️⃣ HTTP üzerinden sayfa tarayıcıya gönderilir

👤 Client (İstemci)

İstek yapan taraftır. Web tarayıcısı bir client'tır.

🖥️ Server (Sunucu)

İsteği yanıtlayan taraftır. Web sunucusu sayfaları gönderir.

🐘 PHP Nedir?

PHP: Hypertext Preprocessor, sunucu taraflı web programlama dilidir. Açık kaynaklı ve ücretsizdir. HTML ile birlikte kullanılabilir. Veritabanı ile kolay entegrasyon sağlar.

⚠️ ÖNEMLİ:

PHP kodları tarayıcıya gönderilmez, sunucuda çalışır! Kullanıcı PHP kaynak kodunu göremez!

📌 Değişken Kuralları

  • Değişken isminin önüne $ işareti konulur
  • Türkçe karakter kullanılmamalıdır (ğ, ü, ş, ı, ö, ç ❌)
  • _ (alt çizgi) ile başlayabilir, rakam ile başlayamaz
  • _ dışında özel işaret kullanılamaz
  • Büyük-küçük harf önemlidir ($toplam ≠ $Toplam)

✅ Doğru Kullanımlar

$deger = 2;
$deger_2 = "GOP";
$Deger21 = 5.55;
$_sayi = 100;
$userName = "Ahmet";

❌ Yanlış Kullanımlar

$2deger  ← rakamla başlar
$deger?  ← ? kullanılamaz
$değer   ← ğ Türkçe karakter
$1sayi   ← 1 ile başlar
$isim%   ← % kullanılamaz
$maaş    ← ş Türkçe karakter

🔍 isset()

🫙 empty()

↪️ header()

Sayfa yönlendirme

💡 Özet:

isset() → değişken var mı? | empty() → değişken boş mu? | header() → sayfayı yönlendir. header() ekrana bir şey yazdırmadan önce kullanılmalıdır!
