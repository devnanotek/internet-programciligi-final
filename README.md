## Icerik

| Sayfa | Aciklama |
|-------|----------|
| `www/index.html` | **Ana Sayfa** - 16 konu basliginin tamami, kod ornekleri ve aciklamalar |
| `www/calisma.html` | **Calisma Sayfasi** - Pratik PHP alistirmalari ve cozumlu ornekler |
| `www/soru-cevap.html` | **Soru-Cevap** - Hoca sorulari, cozumleri ve slayt hatalarinin duzeltmeleri |
| `www/test.html` | **Test** - 20 soruluk interaktif bilgi testi (dogru/yanlis, coktan secmeli, bosluk doldurma) |

## Nasil Kullanilir?

1. `www` klasorundeki herhangi bir `.html` dosyasina cift tiklayin
2. Tarayicinizda hemen acilir, hicbir kurulum gerekmez
3. Tum sayfalar birbirine linklidir, ust menuden gezin
